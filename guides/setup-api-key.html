Connect Your AI Brain

Tinky needs an AI provider to work its magic. We recommend OpenRouter for the best experience, but you have choices.

1

🤖
Choose Your Provider

Tinky is flexible and supports multiple AI engines. Choose the one that fits your needs.

OpenRouter

Recommended. Access to Claude, GPT-4, Llama, and hundreds more through one key.

OpenAI

Direct integration for ChatGPT models (GPT-4o, GPT-4-mini).

Gemini

Google's powerful native models (Gemini 1.5 Pro/Flash).

Ollama

Local, private AI. Run models on your own hardware for free.

2

🔑
Get Your API Key

For this guide, we'll use OpenRouter. It's the easiest way to get started and offers many free models.

  1. Go to the OpenRouter Keys page.
  2. Sign in or create an account.
  3. Click "Create Key". Give it a name like "Tinky".
  4. Copy the generated key. Keep it secret!
Get OpenRouter Key ↗
Using Ollama? You don't need an API key! Just ensure your local Ollama server is running.
3

⚙️
Add it to Tinky

Now, let's link your key to Tinky so it can start assisting you.

  1. Open the Tinky Dashboard.
  2. Click the AI Provider button (or the Gear icon) in the top right.
  3. Select your provider from the dropdown menu.
  4. Paste your API key into the secure field.
  5. Choose your preferred model by clicking one of the fields below.
  6. Click Save & Apply.
🎉 You're all set! Try selecting some text and pressing Ctrl + ` (backtick will be under esc button) to test it out.
